# 一、设计目标

## 产品目标

帮助全球自由职业者完成：

```text
创建Invoice
    ↓
发送客户
    ↓
付款追踪
    ↓
自动催款
    ↓
收入统计
```

形成完整收款闭环。

---

## 设计原则

### 极简

用户应在30秒内完成发票创建。

---

### 专业

生成符合国际标准的Invoice。

---

### 自动化

减少人工操作。

---

### 全球化

支持多语言、多币种、多国家税务规则。

---

### SaaS化

支持长期订阅收费模式。

---

# 六、数据库设计

---

# users

```sql
CREATE TABLE users (
 id UUID PRIMARY KEY,
 email VARCHAR(255),
 password_hash TEXT,
 name VARCHAR(100),
 company_name VARCHAR(255),
 logo_url TEXT,
 created_at TIMESTAMP
);
```

---

# clients

```sql
CREATE TABLE clients (
 id UUID PRIMARY KEY,
 user_id UUID,
 company_name VARCHAR(255),
 contact_name VARCHAR(255),
 email VARCHAR(255),
 country VARCHAR(100),
 vat_number VARCHAR(100),
 notes TEXT,
 created_at TIMESTAMP
);
```

---

# invoices

```sql
CREATE TABLE invoices (
 id UUID PRIMARY KEY,
 user_id UUID,
 client_id UUID,
 invoice_number VARCHAR(50),
 amount DECIMAL,
 currency VARCHAR(10),
 status VARCHAR(20),
 due_date DATE,
 created_at TIMESTAMP
);
```

---

# invoice_items

```sql
CREATE TABLE invoice_items (
 id UUID PRIMARY KEY,
 invoice_id UUID,
 description TEXT,
 quantity INT,
 unit_price DECIMAL
);
```

---

# payments

```sql
CREATE TABLE payments (
 id UUID PRIMARY KEY,
 invoice_id UUID,
 amount DECIMAL,
 payment_method VARCHAR(50),
 paid_at TIMESTAMP
);
```

---

# reminders

```sql
CREATE TABLE reminders (
 id UUID PRIMARY KEY,
 invoice_id UUID,
 send_time TIMESTAMP,
 status VARCHAR(20)
);
```
